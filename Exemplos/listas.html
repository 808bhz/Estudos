Trabalhando com listas

Listas são estruturas úteis para a criação de relação de itens.

Lista não-ordenada

Lista em que a ordem dos itens não importa.

Exemplo:

  • Teclado
  • Mouse
  • Webcam

Lista ordenada

Lista em que a ordem dos itens importa.

Exemplo:

  1. Ligar o computador
  2. Digite usuário/senha
  3. Aguarde o SO carregar
  4. Procurar e abrir o programa...
